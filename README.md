# eComm-FED-Sandbox
FED build process

## Requirements
- [nodejs](http://nodejs.org/) and [Gulp](http://gulpjs.com/)

## Setup
#### 1. Install gulp globally:
```sh
$ npm install --global gulp
```
#### 2. Install node packages in project directory:
```sh
$ npm install
```
## Start Local Server
```sh
$ gulp
```
